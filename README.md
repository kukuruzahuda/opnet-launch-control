# Launch Control

Premium OPNet launch studio for staged token campaigns, allocation design, and vesting previews.

## Live

- App: https://opnet-launch-control.vercel.app

## Stack

- React
- TypeScript
- Vite
- WalletConnect for OPNet wallet shell

## Notes

- Current version is a premium frontend shell with demo/live contract source handling.
- Built as one of the three flagship apps in the OpenNet portfolio.

## Local

```bash
npm install
npm run dev
```
